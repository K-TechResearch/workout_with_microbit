# Program 説明

## balance.hex

加速度センサで体のフラつきを定量化する

- 利用想定種目
  - 片足立ち
  - プランク
    など
- 使用方法

  1. 計測したい場所に microbit をセットし A ボタンを押す
  2. 1 分間計測
  3. 測定完了後 C（平均振れ幅値）M（最大振れ幅値）が表示される

- 演算方法
  - 計測加速度絶対値と重力加速度値 1023 の差分の絶対値を定期間隔(200ms)で 1 分間測定
  - 1 回あたり測定値の平均を平均振れ幅値としている
  - 計測期間内の最大加速度と重力加速度 1023 の差分絶対値を最大振れ幅値としている
